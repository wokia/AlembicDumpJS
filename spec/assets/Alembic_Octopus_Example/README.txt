The alembic_octopus.mb maya scene is 100 frames of a deforming MFnPolyMesh 
model with 94784 polygons. The alembic_octopus.abc file was 
generated by exporting from using this command:
    AbcExport -j "-root octopus_low -fr 20 50 -file alembic_octopus.abc";
The resulting alembic file is 36MB in size.
